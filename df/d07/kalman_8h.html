Update state vector when there is a measurement. It modifies meas.

xhat3=xhat+M*(meas-Cd*xhat) //xhat is x_k|k-1. xhat3 is x_k|k xhat =Ad*xhat3; //xhat is now x_k+1|k xhat2=AdM*xhat3; //xhat2 is x_k+1/M|k
