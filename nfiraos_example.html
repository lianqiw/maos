MAOS
Multithreaded Adaptive Optics Simulator
TMT NFIRAOS LGS MCAO

The default configuration default.conf uses mcao_lgs.conf which describes the baseline configuration of TMT NFIRAOS. It includes the following files to define all the parameters.

  • sim.conf to define some general parameters. It deletes all previous entries,
  • dm_dual.conf to define the dual deformable mirrors
  • atm/atm_mk13n50p.conf to define the turbulence profiles using the Mauna Kea 13N median seeing
  • wfs_lgs_ttf_tt.conf to define the wavefront sensors which is composed of the following
    • powfs_common.conf
    • powfs_none.conf
    • powfs_shlgs.conf
    • llt_CL.conf
    • powfs_shttf.conf
    • powfs_shtt.conf
  • fov_sq34.conf (fit_sq34.conf and evl_sq34.conf) to define the science fov for performance evaluation with IRIS.
  • recon.conf to define the parameters about turbulence reconstruction and DM fitting
  • dbg.conf to define debugging and saving/loading geometry and telemetry.
