When tCoG gradient offset is updated using sodium fit, we need to restrict corrections to fewer zernike modes to avoid unstable. It reuses twfs mode parameters. Noise weighting is not used as it doesn't operate in instantenous gradients. Create reconstruction parameters that are related to the geometry only, and will not be updated when estimated WFS measurement noise changes.

This can be used to do NCPA calibration.
