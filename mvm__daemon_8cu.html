Detailed Description

A standalone mvm daemon by gpu_mvm_daemon(). Waiting correction from MAOS, read pixels, and send DM commands back.

This is not part of maos executable.

For RTC benchmarking, use mvmfull_iwfs.cu which does both gradient processing and MVM, in a piplined way.
